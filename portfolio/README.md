# Sivaneswaran Thiruththanikan — Portfolio

This is a Vite + React portfolio scaffold using Tailwind CSS and Framer Motion.

Getting started:

1. Open a terminal in the `portfolio` folder.
2. Install dependencies:

```
npm install
```

3. Initialize Tailwind (if not already handled by postcss/tailwind):

```
npx tailwindcss -i ./src/index.css -o ./dist/output.css --watch
```

4. Run dev server:

```
npm run dev
```

Notes:
- The project uses `react-router-dom` for routing, `framer-motion` for animations, and `tailwindcss` for styling.
- Update the placeholder project links and social URLs in `src/components/Projects.jsx` and `src/components/Contact.jsx`.
